## Usage
#### Command
```
./lsystem < ./examples/algae.txt
```
#### Source format
Contents of `algae.txt`
```
Algae {
  Axiom A
  A = AB
  B = A
}
```

#### Result
```
A
AB
ABA
ABAAB
ABAABABA
ABAABABAABAAB
ABAABABAABAABABAABABA
ABAABABAABAABABAABABAABAABABAABAAB
...
```
(it never stops)

# Generate images

./lsystel < examples/plant.txt | ./turtle_interpretation 20 > tes.svg
